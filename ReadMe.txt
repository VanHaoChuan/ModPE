这是用于在pc上编写ModPE的简易工具
--------------------------------------------------------------------
在LocalCode.js中放置代码.
千万不要动PCRunner.js中的东西.
--------------------------------------------------------------------
放置代码后可在Debugger.html查看.



--------------------------------------------------------------------
纯小白需了解.
首先,登陆
https://code.visualstudio.com/docs/?dv=win
下载Visual Studio Code(大约50MB).
进入后点左边最后一个几个方块叠成的图标.
扩展里搜索JavaScript.
下载第一个,下载完后点重新加载.
然后点文件图标,点开LocalCode.js,写代码.
想运行就右键Debugger.html,在资源管理器中打开,然后在资源管理器中双击Debugger.html.
最后编写完毕就可以改名拿走了.
